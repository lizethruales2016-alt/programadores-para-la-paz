Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

el servidor responde con la siguiente información:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que Express recibe correctamente los datos enviados mediante req.body y los devuelve en formato JSON.
